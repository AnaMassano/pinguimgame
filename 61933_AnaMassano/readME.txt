A versão que desenvolvi do jogo base do pinguin, tem como objetivo encontrar um tesouro.
Para tal, pelo caminho irá ser possível apanhar moedas e antes de encontrar o tesouro é necessário 
encontrar uma chave que o irá abrir.
Durante o jogo é possível disfrutar de uma música de fundo.
Para jogar basta usar as setas de direção do seu teclado ou usar os botões disponíveis na página do jogo.
